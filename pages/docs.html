Connection

Server URL: https://govcon.ezbizservices.com/mcp

Auth: Pass your API key as X-API-Key header or api_key query parameter.

Tools

search_contracts
Search federal contract opportunities on SAM.gov — filter by NAICS code, set-aside type, agency, dollar range, and keywords. Returns active solicitations with due dates, contact info, and bid requirements.

Parameters:
- query (string): Keywords to search for (e.g., 'cybersecurity', 'IT support', 'janitorial services')
- naics (string, optional): NAICS code to filter by (e.g., '541512' for computer systems design)
- set_aside (string, optional): Set-aside type: 8a, HUBZone, SDVOSB, WOSB, SBA, or 'all'
- agency (string, optional): Federal agency name (e.g., 'Department of Defense', 'VA')

analyze_agency
Analyze a federal agency's spending patterns and contracting behavior — top contractors, spending trends, preferred NAICS codes, contract types, and upcoming opportunities. Uses USASpending.gov data.

Parameters:
- agency (string): Federal agency name (e.g., 'Department of Defense', 'Department of Veterans Affairs', 'GSA')
- fiscal_year (string, optional): Fiscal year to analyze (e.g., '2025')
- naics (string, optional): NAICS code to focus analysis on

competitor_win_analysis
Analyze a competitor's federal contract wins — awarded contracts, agencies they work with, contract values, NAICS codes, and win patterns. Find out who's winning in your space and what they're doing differently.

Parameters:
- company (string): Company name to analyze (e.g., 'Booz Allen Hamilton', 'SAIC')
- naics (string, optional): NAICS code to filter wins by
- agency (string, optional): Agency to filter wins by

set_aside_tracker
Track small business set-aside opportunities across federal agencies — 8(a), HUBZone, SDVOSB, WOSB, and SBA set-asides. Shows trending categories, upcoming deadlines, and agency set-aside spending patterns.

Parameters:
- set_aside_type (string): Set-aside type: '8a', 'HUBZone', 'SDVOSB', 'WOSB', 'SBA', or 'all'
- naics (string, optional): NAICS code to filter by
- agency (string, optional): Agency to filter by

API Endpoints

  • POST /mcp — MCP protocol endpoint
  • POST /api/keys/signup — Create free API key (body: {name, email})
  • GET /api/keys/usage?key=YOUR_KEY — Check usage
  • GET /api/pricing — View pricing tiers
  • GET /health — Health check
